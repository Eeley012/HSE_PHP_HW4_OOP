# Домашнее задание №4 НИС PHP 

## Структура файлов

* **`index.php`** — Главный файл. Запускает проект и демонстрирует работу всех методов.
* **`Good.php`** — Базовый абстрактный класс товара.
* **`GoodInterface.php`** — Интерфейс, определяющий обязательные методы.
* **`PrintTrait.php`** — Трейт для вывода сообщений.
* **`Book.php`** — Класс "Книга" (наследник).
* **`Torch.php`** — Класс "Фонарь" (наследник).
* **`Knife.php`** — Класс "Нож" (наследник).

## Требования
1.  **Наследование**: Классы `Book`, `Torch`, `Knife` наследуются `Good`.
2.  **Абстрактные классы**: Класс `Good` абстрактный, содержит абстрактные методы `getCategory` и `calculateDiscount`.
3.  **Интерфейсы**: Реализован `GoodInterface` (методы `getName`, `getPrice`).
4.  **Трейты**: `PrintTrait` для дублирования логики вывода.
5.  **Конструктор**: Валидация: цена не может быть отрицательной).
6.  **Магические методы**:
    * `__toString` — строковое представление объекта.
    * `__debugInfo` — настройка вывода для `var_dump`.
    * `__clone` — изменение имени при клонировании.
    * `__sleep` / `__wakeup` — управление сериализацией (сохранение и восстановление).
7.  **Ключевое слово Final**: Метод расчета скидки в классе `Book` запрещено переопределять.
8.  **Анонимные классы**: В `index.php` используется анонимный класс для проверки цен.

## Инструкция по запуску
    ```bash
    php -S localhost:8000
    ```
Перейдите на: `http://localhost:8000`
